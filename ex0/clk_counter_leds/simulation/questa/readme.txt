# How to run any of the simulations with the QuestaSim Tool, utilizing TCL (.do) script files:
# 1. Open Questa Tool and then select File -> Change Directory ...  to the 'ex0\clk_counter_leds\simulation\questa' dir 
# 2. Then in the Questa Tool Command Terminal type :  "do run_qsim_tb_1.do" or "do run_qsim_tb_2.do"  - depending on which testbench you want to run.
